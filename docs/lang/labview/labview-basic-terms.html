Basic Terms

Project와 VI를 만들면 아래와 같은 창을 볼 수 있습니다.

프런트패널​

프런트패널은 사용자가 값을 입력하거나 데이터를 확인할 수 있는 창으로, 각종 그래프, 3D 장면 등을 쉽게 구현하여 볼 수 있는 공간입니다.

블록다이어그램​

블록다이어그램은 GUI 방식의 코딩을 통해 알고리즘을 구현하는 공간입니다.

팔레트​

팔레트는 사용자가 클릭하여 사용할 수 있는 도구의 집합으로 프런트패널 팔레트, 블록다이어그램 팔레트 등 여러가지가 있습니다. 프런트패널 팔레트는 프런트패널에서 오른쪽 클릭을 하면 나오고, 블록다이어그램 팔레트는 블록다이어그램에서 오른쪽 클릭을 하면 나옵니다.

터미널​

터미널을 통해 블록다이어그램에서 프런트패널로 데이터를 출력하거나, 또는 프런트패널에서 블록다이어그램으로 데이터를 입력할 수 있습니다. 터미널에는 화살표가 있는데, 주로 터미널을 통해 값이 블록다이어그램에서 프런트패널로 출력되는 경우 화살표가 왼쪽에 있고, 반대로 입력되는 경우 화살표가 오른쪽에 있습니다. 데이터는 왼쪽에서 오른쪽으로 이동한다고 가정하기 때문인데, 사용하다 보면 이해가 되실 것입니다.

컨트롤​

컨트롤을 통해 프런트패널에 사용자가 값을 입력하면 컨트롤에 연결된 터미널을 통해 블록다이어그램으로 전달됩니다. 프런트패널 팔레트에서 고를 수 있고, 자동으로 블록다이어그램에 터미널이 추가됩니다.

인디케이터​

인디케이터를 통해 블록다이어그램에서 터미널을 통해 프런트패널로 보낸 값을 볼 수 있습니다. 프런트패널 팔레트에서 고를 수 있고, 자동으로 블록다이어그램에 터미널이 추가됩니다.

노드​

노드는 블록다이어그램에서 데이터들을 처리하기 위해 사용되는 도구입니다.

익스프레스 VI​

Express VI는 설정 대화 상자 VI, 소스 VI 등으로 구성되 있습니다. 특정 기능을 하나로 묶은 노드라고 생각하시면 됩니다. (해당하는 기능이 있다면) 복잡한 코드를 작성하지 않고도 하나의 기능을 익스프레스 VI 하나로 해결할 수 있습니다. 블록다이어그램 팔레트에서 고를 수 있습니다.
