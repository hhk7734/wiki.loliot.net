&& cd sample

Run debug mode​

flutter run -d linux

Run release mode​

flutter build linux
./build/linux/release/bundle/sample

Build-troubleshoot​

flutter build linux --verbose

Run Without Xwindow(On console mode)​

xinit /path/to/sample/build/linux/release/bundle/sample
