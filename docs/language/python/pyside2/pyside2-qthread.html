QThreadмҷҖ QObjectвҖӢ

Reentrant(мһ¬м§„мһ…) : A class is reentrant if it's safe to use its instances from more than one thread, provided that at most one thread is accessing the same instance at the same time. A function is reentrant if it's safe to invoke it from more than one thread at the same, provided that each invocation references unique data. In other words, this means that users of that class/function must serialize all accesses to instances/shared data by means of some external locking mechanism.

Thread-safe(м“°л Ҳл“ң м•Ҳм „) : A class is thread-safe if it's safe to use its instances from more than one thread at the same time. A function is thread-safe if it's safe to invoke it from more than one thread at the same time even if the invocations reference shared data.

н•ң м“°л Ҳл“ңмқҳ мқҙлІӨнҠё лЈЁн”„лҠ” мқҙлІӨнҠёлҘј н•ҙлӢ№ м“°л Ҳл“ңм—җ мӮҙм•„к°ҖлҠ” лӘЁл“  QObjectм—җ м „лӢ¬н•©лӢҲлӢӨ. QObjectмқҳ м“°л Ҳл“ң м№ңнҷ”лҸ„(thread affinity)лҠ” н•ҙлӢ№ к°қмІҙк°Җ мӮҙм•„к°ҖлҠ” м“°л Ҳл“ңлҘј л§җн•©лӢҲлӢӨ. м“°л Ҳл“ң м№ңнҷ”лҸ„лҠ” QObject::thread()лҘј нҶөн•ҙ м•Ң мҲҳ мһҲмҠөлӢҲлӢӨ. QObjectк°Җ мӮҙм•„к°ҖлҠ” м“°л Ҳл“ңлҘј м ңмҷён•ң лӢӨлҘё м“°л Ҳл“ңм—җм„ң н• лӢ№н•ҙм ңлҘј н•ҳл©ҙ м•Ҳлҗ©лӢҲлӢӨ.(м“°л Ҳл“ң м•Ҳм „) лӢӨлҘё м“°л Ҳл“ңм—җм„ң н• лӢ№н•ҙм ңлҘј н•ҳл Өл©ҙ QObject::deleteLater()лҘј мӮ¬мҡ©н•ҳм—¬ м“°л Ҳл“ң м№ңнҷ”лҸ„м—җ л§һлҠ” мқҙлІӨнҠё нҒҗм—җ н• лӢ№н•ҙм ң мқҙлІӨнҠёлҘј л“ұлЎқн•ҳкі  н•ҙлӢ№ мқҙлІӨнҠё лЈЁн”„м—җм„ң мІҳлҰ¬н•ҳл©ҙ лҗ©лӢҲлӢӨ.

QThread мһҗмӢ мқ„ л¶ҖлӘЁлЎң н•ҳлҠ” к°қмІҙлҘј QThread лӮҙм—җ л§Ңл“Ө мҲҳ м—ҶмҠөлӢҲлӢӨ. QThread к°қмІҙлҠ” лӢӨлҘё м“°л Ҳл“ңм—җ мӮҙм•„к°Җкі  мһҲкё° л•Ңл¬ёмһ…лӢҲлӢӨ.

QThread к°қмІҙлҘј н• лӢ№н•ҙм ңн•ҳкё° м „м—җ лӮҙл¶Җм—җ мӮҙм•„к°ҖлҠ” лӘЁл“  к°қмІҙк°Җ лЁјм Җ мӮ¬лқјм ём•ј н•©лӢҲлӢӨ. мқҙлҠ” QThread::run() мқҳ мҠӨнғқм—җм„ң н•ҙлӢ№ м“°л Ҳл“ңм—җ мӮҙм•„к°ҖлҠ” лӘЁл“  к°қмІҙлҘј мғқм„ұн•ҳл©ҙ мүҪкІҢ кө¬нҳ„н•  мҲҳ мһҲмҠөлӢҲлӢӨ.

  • queued connection : мҲҳмӢ мёЎ м“°л Ҳл“ңмқҳ мқҙлІӨнҠё нҒҗм—җ мқҙлІӨнҠёк°Җ м¶”к°Җлҗҳкі , лӮҳмӨ‘м—җ мқҙлІӨнҠё лЈЁн”„м—җм„ң л№ м§Ҳ л•Ң мҠ¬лЎҜ нҳём¶ң

  • automatic connection : мӢңк·ёл„җ л°ңмғқ м“°л Ҳл“ңмҷҖ мҠ¬лЎҜ мӢӨн–ү м“°л Ҳл“ңлҘј л№„көҗн•ҳм—¬ direct or queued connection

QThread нҢҢмғқ нҒҙлһҳмҠӨ AлҘј л§Ңл“Ө л•Ң, лӮҙл¶Җм—җ мӢңк·ёл„җкіј мҠ¬лЎҜмқ„ л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ. к·ёлҹ° кІҪмҡ° мӢңк·ёл„җкіј мҠ¬лЎҜмқҖ мң„ к·ёлҰјм—җм„ң л№Ёк°„мғү мӣҗм—җ мң„м№ҳн•ҳкІҢ лҗ©лӢҲлӢӨ.

Mainмқҳ мӢңк·ёл„җмқ„ Aмқҳ мҠ¬лЎҜм—җ м—°кІ°н•ҳл©ҙ direct connectionмқҙ лҗ©лӢҲлӢӨ. Aмқҳ мҠ¬лЎҜмқҖ Main event loopм—җм„ң мӢӨн–үлҗ©лӢҲлӢӨ. A м“°лһҳл“ң к°қмІҙмқҳ ліҖмҲҳлҠ” Main event loopм—җм„ң мӢӨн–үлҗҳлҠ” Aмқҳ мҠ¬лЎҜм—җм„ң м ‘к·јмқҙ к°ҖлҠҘн•ҳкі , A event loopм—җм„ңлҸ„ лҸҷмӢңм—җ м ‘к·јмқҙ к°ҖлҠҘн•ҳкё° л•Ңл¬ём—җ л¬ём ңк°Җ л°ңмғқн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

QThreadмқҳ нҢҢмғқ нҒҙлһҳмҠӨм—җ мҠ¬лЎҜмқ„ мғқм„ұн•ҳл©ҙ direct connectionм—җ мқҳн•ң л¬ём ңк°Җ л°ңмғқн•  нҷ•лҘ мқҙ лҶ’кё° л•Ңл¬ём—җ мЎ°мӢ¬н•ҙм•јн•©лӢҲлӢӨ.

Aмқҳ мӢңк·ёл„җмқҙлӮҳ мҠ¬лЎҜмқҖ Main лҳҗлҠ” A event loop м–ҙл””м—җм„ңлӮҳ мӢӨн–үлҗ  мҲҳ мһҲмҠөлӢҲлӢӨ. м—°кІ° мў…лҘҳлҘј кІ°м •н•ҳлҠ” кІғмқҖ мӢңк·ёл„җкіј мҠ¬лЎҜмқҳ мң„м№ҳк°Җ м•„лӢҲлқј мӢӨн–үлҗҳлҠ” loopк°Җ к°ҷмқҖм§Җ лӢӨлҘём§Җм—җ л”°лқј кІ°м •лҗ©лӢҲлӢӨ.

Mainмқҳ мҠ¬лЎҜкіј Aмқҳ мӢңк·ёл„җмқ„ м—°кІ°н–Ҳмқ„ л•Ң, Aмқҳ мӢңк·ёл„җмқҙ нҢҢлһҖ мӣҗм—җм„ң л°ңмғқн•ҳл©ҙ direct, мЈјнҷ© мӣҗм—җм„ң л°ңмғқн•ҳл©ҙ queuedк°Җ лҗ©лӢҲлӢӨ.

QTм—җм„ң мқҳлҸ„н•ҳм§Җ м•ҠмқҖ мӮ¬мҡ©л°©лІ•мқј мҲҳ мһҲмҠөлӢҲлӢӨ. м¶”нӣ„ кІҖнҶ н•ҙм„ң мҲҳм •н•ҳкІ мҠөлӢҲлӢӨ.

import sys

from PySide2.QtWidgets import QMainWindow, QApplication
from PySide2.QtCore import QThread, Slot, Signal

from ui_mainwindow import Ui_MainWindow


class BackgroundThread(QThread):
    signalInA = Signal(str)

    def __init__(self):
        super().__init__()

    def run(self):
        while True:
            self.msleep(1000)
            """
            signal : A event loop -> main event loop
            queued connection
            """
            self.signalInA.emit("A event loop")


class MainWindow(QMainWindow, Ui_MainWindow):
    _backgroundThread = BackgroundThread()

    def __init__(self):
        super().__init__()

        self.setupUi(self)
        self.pushButton.setText("лІ„нҠј 1")
        self.pushButton2.setText("лІ„нҠј 2")

        self.pushButton.clicked.connect(self.pushButtonClicked)
        self.pushButton2.clicked.connect(self.pushButton2Clicked)

        self._backgroundThread.signalInA.connect(self.appendTextBrowser)

        if not self._backgroundThread.isRunning():
            self._backgroundThread.start()

    def pushButtonClicked(self):
        """
        м“°л Ҳл“ңк°Җ мӢӨн–ү мӨ‘мқҙм§Җ м•ҠлӢӨл©ҙ мӢӨн–ү
        """
        if not self._backgroundThread.isRunning():
            self._backgroundThread.start()

    def pushButton2Clicked(self):
        """
        signal : main event loop -> main event loop
        direct connection
        """
        self._backgroundThread.signalInA.emit("main event loop")
        """
        л¬ём ң terminate мӮ¬мҡ©мңјлЎң мқён•ң л¬ём ңк°Җ л°ңмғқн•  мҲҳ мһҲмқҢ.
        """
        self._backgroundThread.terminate()

    @Slot(str)
    def appendTextBrowser(self, text):
        """
        main event loopм—җм„ң мӢӨн–үлҗҳлҠ” мҠ¬лЎҜ
        """
        self.textBrowser.append("signal from " + text)


if __name__ == "__main__":
    app = QApplication(sys.argv)
    mainWindow = MainWindow()
    mainWindow.show()
    app_return = app.exec_()

    """
    мў…лЈҢ м „ м“°л Ҳл“ң к°•м ң мў…лЈҢ
    """
    mainWindow._backgroundThread.terminate()
    sys.exit(app_return)
